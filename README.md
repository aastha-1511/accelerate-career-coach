# accelerate-career-coach


Built an AI-powered platform that provides personalized career tools like resume and cover letter
generators, mock interviews, and skill insights.
◦ Implemented interactive dashboards showing weekly updated skill demand and salary trends using
real-time APIs and serverless functions (Inngest).
◦ Developed an AI Resume and Cover Letter Builder with Gemini API for customizable, job-specific,
ATS-optimized content generation.
◦ Integrated user authentication (Clerk) and responsive UI components (Shadcn UI + Tailwind) for a
seamless and secure experience
